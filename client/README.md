# Getting Started

Let's get started!

## Installation

- Enter the following command to clone this Repo.
```bash
git clone https://github.com/SaadFarhanIdress/motivational-reminders-client
```

- Run this command on the root directory of this Repo.
```bash
npm i 
OR
yarn
```

- Run development server
```bash
npm start
OR
yarn start
```

- Create build folder
```bash
npm build
OR
yarn build
```
